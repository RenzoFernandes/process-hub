# ProcessHub

Plataforma full stack para mapeamento de processos empresariais e visualização hierárquica de fluxos operacionais.

## Tecnologias

- React
- Node.js
- TypeScript
- PostgreSQL
- Docker

## Objetivo

Permitir o gerenciamento de áreas, processos e subprocessos de uma empresa através de uma interface moderna e interativa inspirada em ferramentas como Jira, Linear e Trello.

## Funcionalidades

- Cadastro de áreas
- Gestão de processos e subprocessos
- Visualização hierárquica de fluxos
- Organização de responsáveis, ferramentas e documentações
- API RESTful integrada ao front-end

Frontend → API REST → Banco de Dados
